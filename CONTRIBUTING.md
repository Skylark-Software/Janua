# Contributing to Janua

Thanks for your interest in contributing!

## How to Contribute

1. **Report Bugs** - Open an issue with steps to reproduce
2. **Request Features** - Open an issue describing your use case
3. **Submit PRs** - Fork, branch, make changes, open a pull request

## Development

```bash
git clone https://github.com/Skylark-Software/Janua.git
cd Janua
cp .env.example .env
docker compose up --build
```

Access the UI at `http://localhost:8085/guacamole`

## Code Style

- Keep changes focused and minimal
- Test your changes before submitting
- Update documentation if needed
